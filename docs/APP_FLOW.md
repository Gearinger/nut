# 此时此地 - 页面与用户流程 (APP_FLOW)

## 1. 用户旅程总览

```
下载 → 注册 → 首页(地图) → 进入围栏/创建围栏 → 空间流/群组 → 互动 → 离开
```

## 3. 核心流程

### 3.1 首次登录流程

```
1. 打开 App
   ↓
2. 启动页（3秒）
   ↓
3. 登录页（手机号/邮箱）
   ↓
4. 权限请求（位置、通知、相机）
   ↓
5. 首页（地图）
```

### 3.2 进入围栏流程

```
1. 首页地图显示周围围栏
   ↓
2. 用户点击围栏
   ↓
3. 围栏详情页（显示名称、人数、简介）
   ↓
4. 需要密码？→ 输入密码 → 验证
   ↓
5. 进入围栏空间流
   ↓
6. 可发送消息/帖子
```

### 3.3 创建围栏流程

```
1. 首页点击 "+" 按钮
   ↓
2. 选择创建围栏
   ↓
3. 设置围栏信息：
   - 名称（必填）
   - 简介（选填）
   - 位置（地图选点）
   - 半径（100m - 1km）
   - 时间（开始/结束时间）
   - 密码（可选）
   - 是否公开
   ↓
4. 提交审核（AI + 人工）
   ↓
5. 审核通过，围栏生效
```

### 3.4 发送帖子流程

```
1. 在围栏内
   ↓
2. 点击底部 "+" 按钮
   ↓
3. 选择帖子类型（文字/图片/视频）
   ↓
4. 编辑内容
   ↓
5. 预览 → 发布
   ↓
6. 帖子出现在空间流
```

## 5. 状态管理

### 5.1 用户状态

| 状态 | 描述 |
|------|------|
| 未登录 | 未注册/登录 |
| 已登录 | 正常用户 |
| 信用受限 | 违规导致限制 |
| 围栏内 | 当前在某个围栏中 |
| 围栏外 | 不在任何围栏中 |

### 5.2 围栏状态

| 状态 | 描述 |
|------|------|
| 审核中 | 等待 AI/人工审核 |
| 进行中 | 围栏已生效 |
| 已过期 | 时间已到 |
| 已关闭 | 创建者主动关闭 |
